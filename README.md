**Uber Data Analysis Dashboard** <br>
This repository contains a Power BI dashboard for Uber data analysis, designed to provide comprehensive insights into Uber’s ride and vehicle data. The dashboard helps users understand operational performance, revenue trends, rider demographics, vehicle utilization, and location analytics.

**Features** <br>
Interactive Visuals: Engaging visuals for clear data storytelling, including icons, modern layouts, and navigation buttons.

Tabbed Navigation: Quickly switch between Home, Overview, Vehicle, Revenue, Rider, and Location dashboards for a focused view.

Summary Metrics: At-a-glance KPIs to understand key business metrics.

Category Analysis: Explore data from multiple perspectives such as vehicle details, rider statistics, and revenue breakdowns.

Location Insights: Geographical analysis to pinpoint trends and hotspots.

**Dashboard Sections** <br>
Home: Landing page with navigation and project branding for easy access to dashboard sections.

Overview: Presents summary metrics and key high-level insights for a quick understanding of Uber’s operations.

Vehicle: Focuses on analysis of vehicle types, ride counts per vehicle, and related statistics.

Revenue: Visualizes trends in Uber’s revenue, allowing exploration by periods and categories.

Rider: Provides insights into Uber riders, covering user distribution, ride frequency, and demographic patterns.

Location: Analyzes location-based data, including Total distance, timeslots, distance by location & timeslot.
